# 🚀 CODSOFT Data Science Internship

This repository showcases the machine learning projects I completed during my **CodSoft Data Science Internship**. Each project follows a structured end-to-end machine learning workflow, including data preprocessing, exploratory data analysis, feature engineering, model development, evaluation, documentation, and an interactive project showcase.

## 🌐 Live Project Showcase

**Live Website:** https://anshul-verma20.netlify.app

Explore all completed internship projects through a modern interactive landing page, with direct access to each project's documentation, methodology, and implementation.

---

## 📂 Projects

### ✅ Task 1 – Titanic Survival Prediction

A complete binary classification project to predict passenger survival on the Titanic dataset.

**Highlights**
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Multiple Classification Models
- Hyperparameter Tuning
- Model Evaluation
- Prediction Pipeline
- Interactive Project Landing Page

---

### ✅ Task 2 – Movie Rating Prediction

A regression-based machine learning project for predicting IMDb movie ratings using metadata and feature engineering.

**Highlights**
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Multiple Regression Models
- Hyperparameter Tuning
- Performance Evaluation (R², RMSE, MAE)
- Prediction Generation
- Interactive Project Landing Page

---

### ✅ Task 5 – Credit Card Fraud Detection

An end-to-end fraud detection system for identifying fraudulent credit card transactions using highly imbalanced financial data.

**Highlights**
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Class Imbalance Handling
- Multiple Classification Models
- Hyperparameter Tuning (GridSearchCV)
- Model Evaluation (Accuracy, Precision, Recall, F1-Score, ROC-AUC)
- Fraud Prediction Pipeline
- Automated Visualization Generation
- Interactive Project Landing Page
